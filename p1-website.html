My Portfolio

Reflection

Creating a website showed me the full process of designing and developing a working webpage. This website is my portfolio where I show all the projects I will make while coding. I wanted my website to be minimalist and easy to navigate. Before beginning on the website with VSS code I came up with many wireframes for my home page with a header, title, an about me and a showcase of my portfolio. Firstly, the header element and title refused to align correctly to where it was in the header is above and not to the side of the title, this was fixed as I realised that the title was accidentally in the header this the code recognising this the title was moved to be under the header. Both elements were then moved via flexbox after the title was removed because the header was stacked above the title. The next problem on the home page was moving the preview of the portfolio to the right of the about me instead of under it, this was fixed as both were flexboxes thus I just had to change the flex direction to row instead of column by adding another class attribute (flex-format) which reset the flex direction. The next problem was the padding between all boxes being too close making the website feel cramped; this was fixed by using the padding element and adjusting all flex boxes. Next was the portfolio page which was quite easier than the home page due to learning off making the home page. For example the background image was changed and the size was readjusted to fit the reflection next to it. The reflection and preview for projects 2 to 4 were all copied from preview for the portfolio with the image being swapped for the text for the reflection and for the projects the format to be restructured to fit 3 displays. Throughout this project I revised on my HTML skills learnt in class such as all the divs and learnt a lot on with CSS such as how padding worked and how to add a background image in a flexbox. I would like to change the format of the portfolio due to it not being exactly to my liking and eventually have this accessible on all devices.
